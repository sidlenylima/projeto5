Projeto Sprint 5
Descrição

Este projeto é um aplicativo web desenvolvido com Streamlit para análise de dados de veículos.

O objetivo do aplicativo é permitir a visualização interativa dos dados por meio de gráficos e filtros, facilitando a análise das informações.

Funcionalidades
Visualização de histogramas
Gráficos interativos com Plotly
Análise de dados de veículos
Interface web simples e intuitiva
Tecnologias utilizadas
Python
Streamlit
Pandas
Plotly Express
